小白

第一步 有一个远程仓库，gitee,github,gitlab

第二步 获取密钥 先去 c盘 用户 lenovo 找 .ssh 文件

​ 删除里面的内容。

第三步 打开git bash 输入 ssh-keygen -t rsa 会生成文件

返回第二步 去.ssh文件找 rsa.pub 复制全部，这就是你的密钥

![image-20231023193009521](C:\Program Files\nodejs\node_global\hexo-blog\source_posts\小白\image-20231023193009521.png)

放这里面。

第四步 去创建一个仓库

复制链接

第五步 把远程仓库克隆到 你写项目的文件夹里，和整个项目并列

git clone https://gitee.com/yym7018/gitaa.git (注意 ！！！复制的是SSH的链接)

第六步 关联本地工程到远程仓库 （在你克隆的文件夹里执行命令）

git remote add origin https://gitee.com/yym7018/gitaa.git

如果报错说明已经关联了一个远程库，直接删除他

git remote -v (这是查看) 然后删除 git remote rm origin

再次关联 git remote add origin https://gitee.com/yym7018/gitaa.git

第七步 把你的项目文件夹托克隆的文件夹里面

添加目录 添加所有文件（add）//这是添加到缓存

git commit -m “提交” （commit）这是到本地仓库

最后一步 推到远程仓库

git push -u origin master

（git push origin wang 如果有分支的情况下）

ok

//搜索所有分支

git branch

//添加新分支

git branch wang ；然后wang分支就好了 远程仓库刷新页面

//删除分支

git branch -d wang

//切换分支

git checkout wang
